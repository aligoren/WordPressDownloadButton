Download buttons for your blog. Colorful buttons easy to use.

== Description ==
For download links to blogs with this plugin will look even more beautiful. Use this plugin is very easy to have 20 colored buttons. In addition, this plug-in supports English and Turkish languages??.

Supported colors: blue, crimson, red, wellread, green, aqua, silver, gray, orange, azure, pink, purple, yellow, icecold, ghost, eclipse, cyprus, chocolate, celtic, apache

Very easy using:

```[download link="http://johndoe.com/file.zip" color="crimson"]```
